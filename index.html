How do I create a Google Cloud Project & OAuth Client ID?
  1. Go to the Google Cloud Console.
  2. Create a new project (e.g., "LiveAccess Sheet Searcher").
  3. Go to APIs & Services > OAuth Consent Screen, configure as External or Internal, input basic developer details, and add the scope .../auth/spreadsheets.readonly (Google Sheets API Read-only). Add your own email under "Test Users" if your app status is in Testing mode.
  4. Go to APIs & Services > Credentials, click Create Credentials and select OAuth Client ID.
  5. Set application type to Web application.
  6. Under Authorized JavaScript origins, add:
    • http://localhost
    • http://localhost:5000
    • http://127.0.0.1
    • http://127.0.0.1:5000
    • (Or the actual URL hosting the web page)
  7. Click Create, then copy the generated Client ID and paste it above!
  8. Make sure the Google Sheets API is enabled on your Google Cloud project (APIs & Services > Library > search "Google Sheets API" > Enable).
